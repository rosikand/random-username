Random Username Generator

This is an admittedly very simple program which generates a random username for you to use when making accounts. It works similar to how Xbox generates users' default gamertags. More definitively, a random adjective is selected from a giant list which is followed by a random noun and a number between 1 and 99. Enjoy!

How to use it: simply press the button below and it will generate you a few usernames!
